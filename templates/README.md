# Templates

These are template files for setting up your ClawLite workspace.

## Usage

Copy these files to your `workspace/` directory:

```bash
cp templates/*.md workspace/
```

Or manually copy individual files as needed.

## Files

| File | Purpose |
|------|---------|
| `SOUL.md` | Bot persona and communication style |
| `AGENTS.md` | Bot rules and behavior guidelines |
| `TOOLS.md` | Local tool notes and configurations |

## Customization

Edit the files in `workspace/` to customize your bot:

- **SOUL.md** — Give your bot a unique personality
- **AGENTS.md** — Define what to remember and how to behave
- **TOOLS.md** — Document servers, APIs, and common commands

The workspace files are gitignored by default to keep your customizations private.
